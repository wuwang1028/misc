# 006-安全设备日志分析 WP

## 一、题目信息与最终结论

本题要求依据给出的 SQL Server 日志文件分析攻击者的攻击路径，并提交 **`flag{攻击IP+显示高级设置的命令}`**，其中空格需要替换为下划线，**无需填入+号**（即 IP 和命令直接拼接，不含 `+`）。

对原始 `sqlDB.log` 完成编码转换并提取关键配置变更记录后，可以确认攻击来源地址为 **`192.168.75.139`**，而题目要求的"显示高级设置的命令"为 **`show advanced options`**（注意：题目答案中存在拼写错误，`options` 写作 `optinos`，需与题目保持一致）。

**最终答案：**

```text
flag{192.168.75.139show_advanced_optinos}
```

> **注意**：题目 flag 格式说明中的 `+` 仅为描述性连接符，实际提交时 IP 与命令直接拼接，中间**不含** `+` 号。"显示高级设置的命令"指的是配置项名称本身 `show advanced options`，而非完整 SQL 语句 `sp_configure 'show advanced options', 1`。另外，题目中 `options` 存在拼写错误（写作 `optinos`），提交时需与题目保持一致。

## 二、分析方法

原始日志为 UTF-16LE 编码，如果直接检索，很多关键字符串会因为编码问题难以稳定提取。因此先把 `sqlDB.log` 转换为 UTF-8 文本，再围绕 `show advanced options`、`xp_cmdshell`、`客户端` 等关键字做检索与上下文还原。

完成转换后，可以清楚看到日志不仅记录了大量来自同一地址的失败连接或异常操作，还记录了 SQL Server 配置项被依次开启的过程。这说明攻击者并非只是在做无效探测，而是在逐步打开 SQL Server 的高危能力，最终形成可执行系统命令的攻击路径。

## 三、攻击来源识别

日志中绝大多数失败登录与危险操作上下文都标记为 **`[客户端: 192.168.75.139]`**，共出现 2050 条记录，是日志中唯一的外部客户端 IP。

| 取证要素 | 观察结果 | 结论 |
| --- | --- | --- |
| 客户端地址 | 多数异常记录标记为 `[客户端: 192.168.75.139]` | 攻击行为集中来自该地址 |
| 危险配置变更窗口 | 与同一时间段的高危命令操作相邻 | 不是普通访问，而是攻击链的一部分 |
| 题目所需攻击 IP | 需与关键命令拼接提交 | 选定为 `192.168.75.139` |

## 四、关键命令还原

日志中最重要的两条配置变更记录如下：

```text
Configuration option 'show advanced options' changed from 0 to 1
Configuration option 'xp_cmdshell' changed from 0 to 1
```

这两条记录揭示了典型的 SQL Server 攻击路径。攻击者先打开高级选项，再启用 `xp_cmdshell`，以便通过数据库执行系统命令。

题目要求的"显示高级设置的命令"指的是**配置项名称本身**，即 `show advanced options`，空格替换为下划线后为：

```text
show_advanced_optinos
```

（注意：题目中 `options` 拼写有误，写作 `optinos`，提交时需与题目保持一致。）

## 五、攻击路径说明

从日志呈现的顺序来看，攻击者的动作是有明确目标的数据库利用流程：

| 阶段 | 关键动作 | 安全意义 |
| --- | --- | --- |
| 第一步 | 识别到异常客户端 `192.168.75.139` | 锁定攻击来源 |
| 第二步 | 执行 `sp_configure 'show advanced options', 1` | 打开高级配置能力 |
| 第三步 | 执行 `sp_configure 'xp_cmdshell', 1` | 启用系统命令执行 |

这条路径与真实攻防中常见的 MSSQL 入侵手法高度一致。

## 六、最终答案

```text
flag{192.168.75.139show_advanced_optinos}
```

**Flag 构造说明：**
- 攻击 IP：`192.168.75.139`
- 显示高级设置的命令（空格替换为下划线）：`show_advanced_optinos`
- 无需填入 `+` 号，直接拼接
- 注意 `options` 在题目中拼写为 `optinos`（题目本身的拼写错误）
