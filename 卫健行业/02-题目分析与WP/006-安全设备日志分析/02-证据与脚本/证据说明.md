# 006-安全设备日志分析证据说明

本题取证的核心是从 SQL Server 日志中同时提取攻击来源地址与关键配置变更命令。当前阶段性分析表明，日志中的异常连接与危险操作主要来自客户端 `192.168.75.139`，并且数据库配置项被依次开启，形成了清晰的攻击链。

## 关键日志结论

```text
[客户端: 192.168.75.139]
Configuration option 'show advanced options' changed from 0 to 1
Configuration option 'xp_cmdshell' changed from 0 to 1
```

## 结论摘要

| 要素 | 结论 |
| --- | --- |
| 攻击来源 IP | `192.168.75.139` |
| 题目要求的关键命令 | `sp_configure 'show advanced options', 1` |
| 下划线替换后字符串 | `sp_configure_'show_advanced_options',_1` |
| 最终答案 | `flag{192.168.75.139+sp_configure_'show_advanced_options',_1}` |

## 说明

题目要求的是“显示高级设置的命令”，而不是完整的整条命令链。因此当前采用最核心、最直接对应日志含义的命令主体 `sp_configure 'show advanced options', 1`。若题库校验更偏好带前缀写法，再考虑替换为 `exec sp_configure 'show advanced options', 1`。
