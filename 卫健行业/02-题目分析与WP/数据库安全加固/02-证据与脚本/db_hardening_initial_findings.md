# 数据库安全加固：初始状态记录

- 题目名称：数据库安全加固
- 实例地址：http://oj-10-30-15-12-45703.adworld.xctf.org.cn/
- 浏览器访问结果：当前直接进入 `ttyd - Terminal` 页面，而非普通题面页。
- 终端状态：页面左上角可见 `login:` 提示，说明在线终端当前可用。
- 题面要点（来自用户提供截图）：需要对数据库进行安全加固，包括修改 root 密码、删除 test 数据库、禁止 root 用户远程登录；HTTP 访问端口为 18080，ttyd 用户名密码均为 `ctf`，终端内提供了可连接 MySQL 3306 的工具。
- 推进策略：先快速登录终端，核对 MySQL 版本、当前 root 登录策略、test 数据库存在性与判题触发方式，再决定是否可直接完成加固取旗。
