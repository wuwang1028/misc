## 关键证据

### 1. CVE-2021-3129 漏洞利用证据

在流量中可以找到 7 条 POST `/_ignition/execute-solution` 请求，其中 2 条返回 200（成功写入），5 条返回 500（探测/失败）。

成功的两条请求均使用了如下 `viewFile` 参数：
```
php://filter/write=convert.iconv.utf-8.utf-16be|convert.quoted-printable-encode|convert.iconv.utf-16be.utf-8|convert.base64-decode/resource=../storage/logs/laravel.log
```

写入 PHAR 载荷的请求（步骤4）中，`viewFile` 包含 UTF-16LE + quoted-printable 编码的 PHAR 文件内容，解码后可见 PHAR 头 `<?php __HALT_COMPILER(); ?>` 及 `Illuminate\Broadcasting\PendingBroadcast` 反序列化对象。

### 2. Godzilla Webshell 证据

`/about.php` 收到 4 条成功的 POST 请求，请求体均为 Base64 字符串，响应体也为 Base64 字符串。

使用 Godzilla PHP_AES 解密参数（密码 `rebeyond`，AES-128-CBC，key/IV 均由 `md5("rebeyond")` 派生）可成功解密所有请求和响应。

### 3. Flag 提取证据

操作 5（第 4 条 about.php POST 请求）的响应解密后得到：
```json
{"status":"c3VjY2Vzcw==","msg":"ZmxhZ3s2YW82Ym5saXllbHBmMm01d3VkbXQ4bGR1ZHRuZ2VyOH0="}
```

其中 `msg` 字段 Base64 解码后为：
```
flag{6ao6bnliyelpf2m5wudmt8ldudtnger8}
```
