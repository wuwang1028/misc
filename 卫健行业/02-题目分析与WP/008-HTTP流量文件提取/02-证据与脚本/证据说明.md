# 008-HTTP流量文件提取证据说明

本题取证的核心并不在 HTTP 请求行本身，而在响应体中恢复出的压缩对象内容。当前分析已经确认，流量中存在一个通过 HTTP 返回的 ZIP 文件对象，并且该 ZIP 内部文件名为 `wstmart.sql`。

## 关键证据

```text
Content-Type: application/zip
PK...wstmart.sql...
```

## 结论摘要

| 要素 | 结论 |
| --- | --- |
| 传输对象类型 | ZIP 压缩文件 |
| 压缩包内关键文件 | `wstmart.sql` |
| 被窃取数据库名称 | `wstmart` |
| 最终答案 | `flag{wstmart}` |

## 说明

题面询问的是“被窃取的是哪个数据库”，因此应提交数据库名本体，而不是完整文件名。`wstmart.sql` 中的 `.sql` 只是导出文件扩展名，去除后得到的 `wstmart` 才是题目要求的最终结果。
