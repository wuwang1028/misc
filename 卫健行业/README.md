# 卫健行业赛题归档

本目录用于统一归档当前已整理完成的卫健行业赛题资料、单题分析材料、取证证据、利用与修复脚本，以及按题目拆分的复现环境方案。

| 顶层目录 | 内容定位 |
|---|---|
| `01-赛题资料/` | 原始与整理后的基础赛题资料，包含总表、附件类题目、题目截图与完整归档包 |
| `02-题目分析与WP/` | 按题目维度归档的正式分析结果，每题单独保存 WP、证据、脚本与研究记录 |
| `03-复现环境/` | 按题目维度归档的复现环境方案，后续逐题补齐 Docker、初始化脚本、修复脚本与验证脚本 |

当前结构已经统一为**“顶层按资料类型分组，二级按题目分组”**的模型。也就是说，题目本身不再直接与“复现环境”这一总类目录并列，而是统一收拢到 `02-题目分析与WP/` 与 `03-复现环境/` 两个大类之下，从而避免单题目录与跨题汇总目录在同一层级造成语义混乱。

目前已完成归档的题目包括 **027-安全漏洞事件分析**、**慢SQL优化**、**主机基线加固**、**系统后门排查** 与 **服务后门排查**；后续新增题目也将遵循同一结构持续补齐。
