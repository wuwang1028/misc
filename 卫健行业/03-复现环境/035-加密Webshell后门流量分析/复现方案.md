### 2.2 核心解密逻辑

本题的核心是识别 **Godzilla PHP_AES 加密模式**并完成解密：

```python
import hashlib
from Crypto.Cipher import AES
from Crypto.Util.Padding import unpad
import base64

password = 'rebeyond'
key_md5 = hashlib.md5(password.encode()).hexdigest()
key = key_md5[:16].encode()   # d41d8cd98f00b204
iv  = key_md5[16:32].encode() # e9800998ecf8427e

def decrypt(data_b64):
    raw = base64.b64decode(data_b64 + '==')
    cipher = AES.new(key, AES.MODE_CBC, iv)
    return cipher.decrypt(raw)
```

## 三、攻击链说明

本题流量还原了一次完整的 **CVE-2021-3129 + Godzilla** 攻击链：

1. 攻击者扫描发现目标运行 Laravel 框架
2. 利用 CVE-2021-3129（Ignition 任意文件写入）通过 `php://filter` 链植入 PHAR 格式 Webshell
3. Webshell 落地于 `/about.php`，使用 Godzilla 工具管理（PHP_AES 模式，密码 `rebeyond`）
4. 攻击者通过 Godzilla 进行目录浏览、路径查询、命令执行等操作
5. 最终执行命令读取 flag
