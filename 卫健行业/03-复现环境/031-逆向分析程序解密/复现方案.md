# 031-逆向分析程序解密：复现方案

本题属于静态逆向分析题，复现重点是还原程序校验逻辑，而不是在黑盒环境中盲猜输入。

## 一、复现目标

从 `a.exe` 中恢复出程序校验的真实 flag。

## 二、推荐步骤

| 步骤 | 操作说明 |
| --- | --- |
| 1 | 识别 `a.exe` 是否存在打包器特征，确认其为 PyInstaller 样本 |
| 2 | 使用 PyInstaller 提取脚本拆包，导出 `CTF.pyc` |
| 3 | 用 Python 3.10 对 `CTF.pyc` 做反汇编，提取函数结构、字符串常量和密文数组 |
| 4 | 识别 `func1`/`func2` 实现为 RC4 的 KSA 与 PRGA |
| 5 | 使用程序中的固定密钥 `flag` 对密文数组做逆向还原 |
| 6 | 将恢复出的 15 字节明文填回 `flag{...}` 形式得到最终答案 |

## 三、预期结果

应恢复出中间明文：

```text
who_find_me_ohh
```

最终提交值为：

```text
flag{who_find_me_ohh}
```
