# 赛题资料目录说明

本目录只保留一份清晰的资料主结构，不再同时保留 `ctf_attachments` 与 `ctf_package_full` 两套重复层级。

| 目录 | 内容 |
|---|---|
| `00-索引与清单/` | 题目整理总表、缺失附件识别记录、索引说明 |
| `01-附件类题目/` | 附件类题目的原始 ZIP 文件，仅保留一份 |
| `02-题目截图/` | 题目截图与辅助图片材料 |
| `99-完整归档包/` | 一份完整打包的总归档文件，便于整体下载备份 |

这样处理后，**单题附件只保留一份原始文件**，而完整压缩包作为整体备份单独保留，既避免重复，又保留一次性下载能力。
